# Objetivos

### Integrar `React.memo`

**Objetivo:** Evitar el re-render de las cartas cuando sus props no cambian.

> Referencia: [React.Memo](https://react.dev/reference/react/memo) – Ej: Una card que se renderice con frecuencia.
>
> Referencia: [useMemo](https://react.dev/reference/react/useMemo) – Ej: Memorizar los estilos de una card que se renderice mucho.

---

### Rutas Suspensas para developers

**Objetivo:** Las rutas para desarrolladores deben suspenderse; un usuario normal no las carga a menos que ingrese explícitamente.

> Referencia: [Suspense](https://react.dev/reference/react/Suspense) – Ej: Evita cargar la UI de rutas específicas hasta que el usuario entre.
